# Immersive Sequence Storyboard

Date: 2026-07-16

## Public narrative role

The sequence represents one idea: SUBEROS connects image, physical output and digital experience inside one visual direction.

## Story beats

1. Intro frame
   - calm poster state;
   - sequence enters as an editorial canvas, not as a fake project reveal.
2. Build
   - geometric forms intensify with scroll;
   - the feeling is coordination, not spectacle.
3. Mid-scroll
   - motion density increases while the copy explains why the immersive layer exists.
4. End state
   - the user exits toward `Trabajo` with the message that all disciplines belong to the same chain of decisions.
5. Reverse scroll
   - the scene scrubs backwards cleanly with the same continuity.

## Copy overlay inside the scene

- Label: `Direccion integrada`
- Title: `Movimiento con funcion narrativa.`
- CTA: `Hablar sobre la experiencia`

## Accessibility rule

If motion is reduced, unsupported or too expensive for the current profile, the poster/fallback image must carry the same message without requiring motion.

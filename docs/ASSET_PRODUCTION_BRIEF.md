# Asset Production Brief

Date: 2026-07-16

## Objective

Prepare only real SUBEROS-owned or explicitly approved assets for the next public portfolio and production launch steps.

## Required packs

1. Branding pack
   - final logo variants if any differ from current approved branding files;
   - favicon confirmation if a new set is required;
   - social share card export if the launch state changes.
2. Portfolio pack
   - discipline-tagged approved images for fotografia, branding, produccion and web;
   - caption and rights status per asset.
3. Sequence pack
   - only if a future public cinematic scene is approved;
   - desktop, tablet and mobile exports plus poster and fallback;
   - publication clearance documented before runtime use.

## Delivery rules

- Prefer WebP for imagery unless transparency or legal source constraints require PNG/SVG.
- Keep original masters outside runtime folders.
- Runtime assets must remain local, optimized and fingerprint-safe.
- Every public asset needs provenance and approval status.
